## Prerequisites

- **🐍 Python3.11**
- [🏠 LM Studio](https://lmstudio.ai/)
- 在LM Studio中下载并加载模型：`deepseek-r1-distill-qwen-32b`
- 确保LM Studio服务器正在运行（默认端口1234）


## 🔨 Application Setup

Create a virtual environment and install all dependencies.

```sh
make setup
```

Activate virtual environment and run:

```sh
playwright install
```


## 🚀 Run the application

```sh
make run
```


## 🛟 Getting Help

```sh
make

# OR

make help
```
